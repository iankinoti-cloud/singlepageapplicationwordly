# Wordly Interactive Dictionary

A single-page application (SPA) for searching English words and exploring their meanings, pronunciations, and more.

## Features

- **Word Search**: Enter a word to get definitions, examples, synonyms, and pronunciations.
- **Audio Playback**: Listen to word pronunciations when available.
- **Favorites**: Save and manage your favorite words locally.
- **Themes**: Switch between day and night themes.
- **Responsive**: Works on desktop and mobile devices.
- **Caching**: Fast lookups with built-in caching.

## How to Run

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Start searching for words!

## Technologies Used

- **HTML5**: Structure and semantics.
- **CSS3**: Styling with custom properties and themes.
- **JavaScript (ES6+)**: Logic, API calls, and interactivity.

## API

Powered by the [Free Dictionary API](https://api.dictionaryapi.dev/).
